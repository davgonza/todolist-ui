To-Do List
===
Normal To-Do List made with Create React App, React and some Redux. 

# [Demo](https://davgonza.github.io/todolist-ui/)

![screen](doc/images/screen.png)


Install app
===

```bash
git clone https://github.com/davgonza/todolist-ui.git
```
install modules
```bash
npm i
```
Start app
===
```bash
npm start
```
